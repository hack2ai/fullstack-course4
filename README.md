## 📋 Assignment Requirements

### ✅ Required (Graded)

| Feature | Description |
|---|---|
| **Navbar** | Scrolls with page (not fixed). Brand name "Food, LLC" aligned left |
| **Desktop / Tablet** | Navbar shows brand only — no menu items visible |
| **Mobile Hamburger** | Toggle button (3 bars) appears on mobile (`visible-xs`) |
| **Mobile Dropdown** | Full-width dropdown with 3 items: Chicken, Beef, Sushi |
| **Page Heading** | "Our Menu" centered using Bootstrap `text-center` |
| **Tall Section** | Full-width Bootstrap grid section (`col-xs-12`) that requires scrolling |

### ⭐ Optional (Bonus)

| Feature | Description |
|---|---|
| **Desktop layout** | 3 equal columns (`col-md-4` each) |
| **Tablet layout** | First 2 sections side-by-side (`col-sm-6`), Sushi full-width (`col-sm-12`) |
| **Mobile layout** | Each section full-width (`col-xs-12`) |
| **Back to Top links** | Each section has a "Back to Top" anchor link |
| **Mobile nav links** | Menu items jump to their respective sections (`#chicken`, `#beef`, `#sushi`) |

## 📱 Responsive Breakpoints

|---|---|---|
| Desktop | `col-md-4` | 3 columns side by side |
| Tablet | `col-sm-6` / `col-sm-12` | 2 cols + 1 full-width |
| Mobile | `col-xs-12` | Single column stacked |
